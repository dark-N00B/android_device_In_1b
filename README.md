Device Tree - Micromax In 1b 64GB (E7533)
===============================================================

Basic   | Spec Sheet
-------:|:-------------------------
CPU     | Octa-core (4x2.3 GHz Cortex-A53 & 4x1.8 GHz Cortex-A53)
Chipset | MediaTek Helio G35 (12 nm)
GPU     | PowerVR GE8320
Memory  | 4 GB
Shipped Android Version | Android 10 (Upgradeable to Android 11)
Storage | 64 GB (eMMC)
MicroSD | Up to 512 GB (dedicated slot)
Battery | Li-Po 5000 mAh, non-removable
Dimensions | 164.5 x 75.8 x 8.9 mm (6.48 x 2.98 x 0.35 in)
Display | 720 x 1600 pixels, 6.52" IPS LCD, 20:9 ratio (~269 ppi density)
Rear Camera  | Dual : 13 MP, f/1.9, 26mm (wide), PDAF; 2MP , f/2.4 (depth)
Front Camera | 	8 MP, f/2.0
Release Month | 2020, November 26
